检测哪些信号

6 项维度，覆盖 TikTok 判定环境的每个破绽

检测原理

TikTok 靠"信号不一致"识别可疑环境

代理能换掉你的 IP，却换不掉手机的时区、语言、指纹和线路质量。我们把 TikTok 会查的信号自己先查一遍，用规则打分——分三类：

01

身份类 · 你这个 IP 干不干净

是机房 IP 还是住宅 IP、ASN 归属、是否被风险库标记为代理、风险评分多高。机房 IP 是养号大忌。

02

一致性类 · 各处身份对不对得上

时区 vs IP 国家、语言 vs IP 国家、WebRTC 暴露的公网 IP vs 代理 IP、UA vs 屏幕。"IP 在美国、时区在上海"就是一眼被识破的破绽。

03

质量类 · 线路能不能扛住发视频

30 秒持续采样 TikTok 域名的延迟、抖动、丢包、TLS 握手。丢包超 2% 就会上传失败、直播掉线。
