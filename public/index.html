检测原理

TikTok 靠"信号不一致"识别可疑环境

代理能换掉你的 IP，却换不掉手机的时区、语言、指纹和线路质量。我们把 TikTok 会查的信号自己先查一遍，用规则打分——分三类：

01

身份类 · 你这个 IP 干不干净

是机房 IP 还是住宅 IP、ASN 归属、是否被风险库标记为代理、风险评分多高。机房 IP 是养号大忌。

02

一致性类 · 各处身份对不对得上

时区 vs IP 国家、语言 vs IP 国家、WebRTC 暴露的公网 IP vs 代理 IP、UA vs 屏幕。"IP 在美国、时区在上海"就是一眼被识破的破绽。

03

质量类 · 线路能不能扛住发视频

30 秒持续采样 TikTok 域名的延迟、抖动、丢包、TLS 握手。丢包超 2% 就会上传失败、直播掉线。

常见问题

FAQ

有。TikTok 在推流前会评估你的网络环境是否「可信」。机房 IP、DNS 与时区不匹配、WebRTC 泄漏真实 IP 等信号，会让算法判定为「高风险环境」，从而限制初始推荐量——表现就是内容不错却 0 播放。
安全。检测全程在你的浏览器本地发起网络请求，不需要登录 TikTok、不碰你的账号密码。我们不存储你的原始公网 IP、真实内网 IP 和 Canvas 指纹原图，只保留匿名派生指标最多 7 天。
90+ 优秀，可放心养号发布；70–89 良好，优化少数项即可；50–69 警告，存在风险，不建议发新视频；50 以下危险，建议立即排查环境再操作。
每次换 IP、换网络、换代理线路后都建议测一次；稳定使用时一周一次即可。检测要用真正发视频的那台手机、那个网络。
机房 / IDC IP 来自数据中心（AWS、Vultr 等），一个 IP 段被大量用户共用，TikTok 极易识别为代理，是养号大忌；住宅 IP 来自真实家庭宽带（如 Comcast），干净、风险低，是养号首选。

隐私声明：本工具不存储你的原始公网 IP、不存 WebRTC 泄漏的真实 IP、不存 Canvas 指纹原图（仅 SHA-256 哈希）。匿名派生指标最多保留 7 天用于分享，之后自动删除，不向第三方出售任何数据。
